## 快速测试步骤

### 方法一：使用两个浏览器窗口（推荐）

1. **获取婚礼ID**
   - 在第一个浏览器（Chrome）中访问：http://localhost:3000
   - 登录你的当前账号
   - 进入任意一个婚礼详情页
   - 在"邀请他人加入"部分，复制6位数字ID（例如：374176）

2. **创建测试用户**
   - 在第二个浏览器（Edge/Firefox）或Chrome的隐私/无痕模式中访问：http://localhost:3000/register
   - 注册新用户：
     - 用户名：`testuser` 或 `user2`
     - 密码：`123456`
     - 邮箱：`test@example.com`（可选）
   - 注册成功后会自动登录

3. **加入婚礼**
   - 在第二个浏览器中，点击"加入婚礼"按钮或访问：http://localhost:3000/weddings/join
   - 输入第一步复制的6位数字ID（例如：374176）
   - 点击"查询"预览婚礼信息
   - 点击"加入婚礼"按钮
   - 应该会自动跳转到婚礼详情页

4. **验证功能**
   - ✅ 新用户应该能看到婚礼的所有任务
   - ✅ 新用户应该能在"我的婚礼"列表中看到这个婚礼
   - ✅ 两个用户都可以更新任务状态

### 方法二：使用浏览器隐私模式

1. **保持当前用户登录**（主窗口）
2. **打开隐私/无痕窗口**（Ctrl+Shift+N 或 Ctrl+Shift+P）
3. **在隐私窗口中注册新用户**
4. **使用新用户加入婚礼**

## 故障排除

如果遇到问题：

1. **服务器未启动**
   - 检查终端是否有错误
   - 访问 http://localhost:3000 确认服务器运行

2. **无法注册新用户**
   - 检查用户名是否已存在
   - 密码至少6位
   - 查看浏览器控制台错误信息

3. **无法加入婚礼**
   - 确认ID是6位数字
   - 确认ID正确（从创建人的婚礼详情页复制）
   - 检查是否已经加入过该婚礼

4. **数据库错误**
   - 运行：`npx prisma generate`
   - 重启服务器
